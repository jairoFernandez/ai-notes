---
title: Producto punto
description: Producto punto entre dos vectores.
---

El producto punto entre dos vectores es una operación matemática que se define como la suma de los productos de los componentes correspondientes de los vectores.

Por ejemplo, si tenemos dos vectores `a = [1, 2, 3]` y `b = [4, 5, 6]`, el producto punto entre ellos es `1 * 4 + 2 * 5 + 3 * 6 = 32`.
